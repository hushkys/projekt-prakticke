# Tipy pro vyhledávání na internetu

Pokud jste v našich návodech nenašli přesně to, co potřebujete, nebo narazíte na specifický problém, nejrychlejší cestou je vyhledat si řešení na internetu. Zde je kategorizovaný seznam ideálních frází (queries), které můžete zadat do vyhledávače (Googlu), abyste našli přesně to, co hledáte.

>  **Tip pro vyhledávání:** Vždy vyhledávejte v angličtině! Komunita v IT je globální a šance, že někdo měl stejný problém jako vy a vyřešil ho na fórech (jako je Stack Overflow, Reddit apod.), je v angličtině téměř 100%.

##  Administrace Windows (Server i Klient)

*   **Instalace Windows 10 ve VirtualBoxu:**
    Pokud si nevíte rady se základní instalací klientského systému:
     *Vyhledejte:* `"how to install windows 10 in using virtualbox"`
*   **Aktivace a konfigurace DHCP ve Windows Serveru:**
    Pro nastavení přidělování IP adres v doméně:
     *Vyhledejte:* `"windows server 2019 dhcp configuration step by step"`
*   **Připojení klientského PC do domény (Active Directory):**
    Pokud vám nejde připojit stanici do nově vytvořené domény (často problém s DNS):
     *Vyhledejte:* `"how to join windows 10 to active directory domain troubleshooting"`
*   **Tvorba sdílené složky a oprávnění:**
    Pro detailní nastavení NTFS a sdílených práv:
     *Vyhledejte:* `"windows server file sharing and ntfs permissions best practices"`
